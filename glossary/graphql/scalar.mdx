title: scalar

import { GlossaryExternalLink } from "@/components/glossary-external-link"

A scalar represents a primitive value in GraphQL and is used to help define the simplest shape of the data that a `field` will return. Scalar types in GraphQL are `String`, `Int`, `Float`, `Boolean` and `ID`. You can also create custom scalar types.
